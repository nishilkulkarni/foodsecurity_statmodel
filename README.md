# Financialization of Agricultural Commodities and Food Security

This repository contains the research paper and supporting materials for the study on the financialization of agricultural commodities and its impact on food security in emerging economies.

## Research Overview
- **Title:** Does the Financialization of Agricultural Commodities Impact Food Security?
- **Authors:** Nishil Kulkarni and Manogna R. L.
- **Scope:** Examines the impact of financial speculation on food price volatility and security in developed and developing nations (2000-2022).
- **Methodology:** Panel data regression analysis and moderating effects model.
- **Key Findings:**
  - Financialization increases food price volatility and negatively affects food security.
  - Post-2008 financial crisis, the impact became more pronounced in emerging economies.
  - Monetary policy plays a role in moderating these effects.

1. Clone the repository:
   ```sh
   git clone https://github.com/nishilkulkarni/foodsecurity_statmodel.git
   ```
2. Navigate to the project directory:
   ```sh
   cd financialization-food-security
   ```
3. Review the research paper (`research-paper.pdf`).
4. Run the analysis scripts (if applicable) from the `scripts/` directory.
